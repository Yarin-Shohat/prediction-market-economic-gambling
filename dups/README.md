# Dups Folder Overview

This folder contains scripts for handling duplicate comments in the project.

## Contents

### add_dup_cols.py
- Adds columns to the main comments dataset to indicate duplicate comment IDs and related metadata.
- Helps track and analyze duplicates for further processing or filtering.

### get_dup_comments.py
- Extracts and processes duplicate comments from the dataset.
- Useful for identifying, reviewing, and handling repeated submissions.

## Usage
- Run these scripts to annotate and extract duplicate comments before classification or analysis.
- All file paths are hardcoded in the scripts.

---
These tools support data cleaning and quality control by ensuring duplicate comments are properly managed.
